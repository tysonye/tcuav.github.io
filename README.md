# UAV 无人机技术网站

一个专业的无人机技术与解决方案提供商网站，使用React、TypeScript和Tailwind CSS开发。

## 快速开始

### 安装依赖
```bash
npm install
```

### 本地开发
```bash
npm run dev
```

### 构建项目
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

### 部署到GitHub Pages
```bash
npm run deploy
```

## 部署说明

本项目已配置GitHub Actions自动部署流程，当代码推送到main分支时，会自动构建并部署到GitHub Pages。

### 主要依赖
- React 18+
- TypeScript
- Tailwind CSS
- Vite
